How it works

Two layers.
Zero compromises.

Guni runs a fast heuristic pass first, then calls the LLM layer only when something looks suspicious.

01 — PARSE
DOM normalization
Raw HTML is parsed. Visible text, hidden elements, forms, scripts and buttons are extracted and normalized.
02 — DETECT
5-vector heuristics
Injection, phishing, UI deception, malicious scripts, and goal mismatch — each checked in parallel in under 1ms.
03 — REASON
LLM intent analysis
When heuristics flag something, Claude reasons about intent — catching reworded and novel attacks no keyword list would find.
04 — DECIDE
Policy enforcement
Risk ≥ 70 → BLOCK. Risk 40–69 → CONFIRM. Risk < 40 → ALLOW. Every decision is logged with full evidence.
API

Integrate in
3 lines.

Drop Guni into any Python agent. Works with LangChain, browser-use, Playwright, AutoGPT — anything that fetches HTML.

python 
from guni import scan

# Drop this before any browser action
result = scan(html=page_html, goal="Login to website")

if result["decision"] == "BLOCK":
    raise SecurityError("Threat detected — action halted")

# result["risk"]       → 0-100 score
# result["evidence"]   → what was found
# result["breakdown"]  → per-category scores
REST API 
# POST /scan
curl -X POST https://your-guni-url/scan \
  -H "Content-Type: application/json" \
  -d '{"html": "<page html>", "goal": "Login to website"}'

# Response
{
  "decision": "BLOCK",
  "risk": 85,
  "evidence": { "injection": ["Visible injection phrase detected"] }
}
